# v1.6.0 — Gameplay depth, cloud saves, trade feedback, and release metadata backfill

Release date: 2026-05-03

This release backfills the major work merged after `v1.5.7`. It packages the cloud-save work, Phase 1/Phase 2 gameplay surfacing, market-depth systems, advanced order tooling, portfolio rebalancing, CI hardening, and release metadata updates into one coherent minor release.

## Highlights

- Added Supabase-backed anonymous cloud saves with local IndexedDB fallback.
- Added Phase 2 trade previews and confirmations for buy, sell, short, and cover actions.
- Added clearer failed-trade messaging with specific reasons and cash/position context.
- Added portfolio-vs-benchmark performance charts with 12-month, 24-month, and full-history views.
- Added portfolio rebalancing by stock or sector target weights, including trade previews and one-click execution.
- Added advanced pending-order tooling: limit orders, stop-losses, and take-profit exits.
- Added watchlists, scheduled catalysts, market pulse panels, upcoming catalyst context, and richer season recaps.
- Added CI, README badges, PR template, and build/test guardrails.
- Stabilized trade feedback exposure math and reduced accidental duplicate trade execution from rapid taps.
- Hardened Game Over routing, completed-run recording, save cleanup, news dedupe, and several regression paths.

## User-facing changes

### Cloud saves

- Saves locally first, then syncs to Supabase when `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` are configured.
- Loads cloud saves first when available, then falls back to local saves.
- Preserves local-only behavior if Supabase environment variables are missing.

### Trade feedback

- Stock detail now shows a live trade preview before executing buy/sell/short/cover actions.
- Successful trades now show confirmation details: cash impact, position impact, exposure, and risk movement.
- Failed trades now explain why the action failed instead of silently feeling inert.
- Quantity/action changes clear stale confirmations and errors.
- Rapid repeated taps are guarded to avoid accidental duplicate execution.

### Portfolio tools

- Portfolio screen now includes a normalized portfolio-vs-benchmark performance chart.
- Portfolio screen now includes an open-orders card for pending limits and protective exits.
- Portfolio screen now includes a rebalancer that accepts signed target weights by stock or sector.
- Rebalance preview shows the required trades before execution.

### Market depth

- Watchlists can surface large moves, fresh news, and upcoming catalysts.
- Scheduled catalyst calendar makes future market events more legible.
- Market Pulse, Upcoming Catalysts, and Season Recap panels add more explainability to each run.

### End-of-run behavior

- Completed runs now record automatically to leaderboard with stable run IDs.
- Final turns now flow into the turn summary and then into Game Over instead of falling back into normal gameplay.
- Completed auto-save slots are cleared so finished games do not appear as resumable progress.
- Completed manual saves are marked clearly and reopen as results.
